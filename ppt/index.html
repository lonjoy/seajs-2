什么是模块化？

把复杂的系统 分拆 成不同模块，并使模块之间通过标准化接口进行 信息沟通 的动态整合过程。

我们为什么需要模块化？

因为有复杂系统 - 拆分

因为有协同工作 - 通信

seajs.config

config.base - 配置基础路径

config.alias - 配置模块别名

config.map - 对文件请求路径进行匹配修改

config.preload - 提前载入必要的插件和模块

seajs.config({
    base: '/sea-modules',
    alias: {
        'jquery': 'gallery/jquery/1.7.2/jquery'
    },
    map:[
        ['a.js', 'a-debug.js'],
        [/^(.*\.(?:css|js))(.*)$/i, '$1?455'] 
    ],
    preload: ['plugin-json', 'plugin-combo']
});
